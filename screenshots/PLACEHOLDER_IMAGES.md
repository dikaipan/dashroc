## Cara Menambahkan Screenshot

1. Ambil screenshot dari aplikasi yang berjalan
2. Simpan dengan nama sesuai daftar di atas
3. Format: PNG (recommended) atau JPG
4. Resolusi: Minimal 1920x1080
5. Refresh browser untuk melihat perubahan

## Auto-Load Feature

Dokumentasi HTML akan otomatis memuat gambar jika file ada di folder ini dengan nama yang sesuai. Sistem akan mencoba beberapa variasi nama file secara otomatis.

## Naming Convention

- Gunakan lowercase
- Pisahkan kata dengan hyphen (-)
- Contoh: `dashboard-overview.png`
- Hindari spasi dan karakter khusus
