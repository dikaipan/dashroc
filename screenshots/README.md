# Folder Screenshots

Folder ini digunakan untuk menyimpan screenshot dari aplikasi ROC Dashboard yang akan digunakan dalam dokumentasi.

## Cara Menggunakan

1. **Ambil Screenshot** dari aplikasi yang berjalan
2. **Simpan screenshot** di folder ini dengan nama yang jelas
3. **Edit file HTML** untuk mengganti placeholder dengan path ke screenshot

## Naming Convention

Gunakan nama file yang jelas dan konsisten:

- `dashboard-overview.png` - Screenshot dashboard overview
- `dashboard-kpi-cards.png` - Screenshot KPI cards
- `dashboard-map.png` - Screenshot peta interaktif
- `engineers-table.png` - Screenshot tabel engineers
- `engineers-form.png` - Screenshot form tambah/edit engineer
- `machines-table.png` - Screenshot tabel machines
- `machines-warranty.png` - Screenshot warranty analytics
- `stockparts-inventory.png` - Screenshot inventory
- `stockparts-fsl-map.png` - Screenshot FSL map
- `structure-chart.png` - Screenshot structure view
- `decision-analysis.png` - Screenshot decision support
- `toolbox-tools.png` - Screenshot toolbox

## Format File

- **Format**: PNG (recommended) atau JPG
- **Resolusi**: Minimal 1920x1080 untuk kualitas baik
- **Ukuran**: Usahakan di bawah 2MB per file untuk performa

## Tips

- Gunakan nama file yang deskriptif
- Simpan dalam format PNG untuk kualitas terbaik
- Kompres gambar jika terlalu besar (>2MB)
- Pastikan screenshot tidak mengandung data sensitif

## Struktur Folder

```
screenshots/
├── README.md (file ini)
├── dashboard-overview.png
├── dashboard-kpi-cards.png
├── engineers-table.png
├── machines-table.png
└── ... (screenshot lainnya)
```
