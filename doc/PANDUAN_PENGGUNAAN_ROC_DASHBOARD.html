1. Pengenalan Sistem

üéØ Tentang ROC Dashboard

ROC Dashboard adalah sistem manajemen terintegrasi untuk mengelola data engineer/teknisi dan mesin customer. Sistem ini dirancang untuk memberikan visibilitas penuh terhadap operasional engineering dan maintenance, dengan fitur analisis mendalam dan decision support untuk optimasi operasional.

üõ†Ô∏è Teknologi yang Digunakan

Kategori Teknologi
Frontend React.js, Recharts, Tailwind CSS, React Feather Icons, Leaflet Maps
Backend Python (Flask), REST API, CSV Data Management
Visualisasi Recharts (Charts), Leaflet (Maps), Interactive Dashboards
üí° Informasi Penting: Sistem ini menggunakan CSV sebagai penyimpanan data, sehingga semua perubahan data akan langsung tersimpan ke file CSV yang sesuai.

2. Fitur Utama

üìä 2.1 Dashboard

üìà Overview & KPI Cards

Dashboard menampilkan ringkasan data penting dalam bentuk KPI (Key Performance Indicators) cards:

  • Total Engineers: Jumlah total engineer yang terdaftar
  • Total Machines: Jumlah total mesin customer
  • Stock Parts: Total spare parts dalam inventory
  • FSL Locations: Jumlah Field Service Location (gudang regional)

üé® Visualisasi Interaktif

  • Peta Interaktif: Distribusi engineer dan mesin berdasarkan lokasi geografis
  • Charts & Graphs: Visualisasi data dalam bentuk bar chart, pie chart, dan line chart
  • Training Progress: Tracking progress training engineer
  • Machine Analytics: Analisis umur mesin, warranty status, dan tren instalasi

üöÄ Cara Menggunakan Dashboard:

1 Akses halaman Dashboard melalui menu sidebar

2 Gunakan filter di bagian atas untuk memfilter data berdasarkan kategori

3 Klik pada KPI cards untuk melihat detail lebih lanjut

4 Klik pada chart untuk membuka tampilan fullscreen

5 Gunakan peta interaktif untuk melihat distribusi geografis

üë• 2.2 Engineers Management

‚ú® Fitur CRUD Lengkap

  • Create: Tambah engineer baru dengan form dinamis
  • Read: Lihat daftar engineer dengan tabel yang dapat di-sort dan di-filter
  • Update: Edit data engineer yang sudah ada
  • Delete: Hapus engineer (dengan konfirmasi)
  • Bulk Delete: Hapus multiple engineer sekaligus

üìù Cara Menggunakan Engineers Management:

1 Klik menu "Engineers" di sidebar

2 Untuk menambah engineer baru: Klik tombol "Add Engineer"

3 Isi form yang muncul, lalu klik "Save"

4 Untuk edit: Klik icon edit (‚úèÔ∏è) pada baris engineer yang ingin diedit

5 Untuk hapus: Klik icon delete (üóëÔ∏è) dan konfirmasi penghapusan

6 Gunakan search box untuk mencari engineer tertentu

7 Gunakan dropdown filter untuk memfilter berdasarkan kategori

8 Klik icon download untuk export data ke CSV

‚öôÔ∏è 2.3 Machines Management

üîß Manajemen Mesin Customer

Halaman ini mengelola semua data mesin customer dengan informasi lengkap tentang status warranty, maintenance, dan informasi teknis lainnya.

üéØ Fitur Utama

  • CRUD Operations: Create, Read, Update, Delete mesin
  • Warranty Tracking: Monitor status warranty mesin
  • Customer Filter: Filter mesin berdasarkan customer
  • Region Filter: Filter berdasarkan region
  • Warranty Filter: Filter berdasarkan status warranty
  • Machine Analytics: Analisis distribusi mesin per customer, region, dan type

üöÄ Cara Menggunakan Machines Management:

1 Klik menu "Machines" di sidebar

2 Gunakan filter di bagian atas untuk memfilter mesin

3 Klik "Add Machine" untuk menambah mesin baru

4 Isi form dengan informasi mesin yang lengkap

5 Klik icon edit untuk mengubah data mesin

6 Klik pada KPI cards untuk melihat analisis detail

üì¶ 2.4 Stock Parts Management

üìä Manajemen Inventory Spare Parts

Halaman ini mengelola inventory spare parts dan Field Service Location (FSL) untuk memastikan ketersediaan parts di setiap gudang regional.

üìù Cara Menggunakan Stock Parts Management:

2 Lihat KPI cards untuk overview stock status

3 Klik "Add Stock Part" untuk menambah parts baru

4 Isi informasi part number, quantity, dan FSL location

5 Monitor stock alerts untuk parts yang perlu perhatian

6 Gunakan peta untuk melihat distribusi FSL locations

3. Cara Penggunaan

üåê 3.1 Akses Sistem

Langkah-langkah Akses:

1 Pastikan backend server sudah berjalan di http://localhost:5000

2 Pastikan frontend server sudah berjalan di http://localhost:5173

3 Buka browser dan akses http://localhost:5173

4 Jika diperlukan, login dengan kredensial yang valid

5 Setelah login, Anda akan diarahkan ke halaman Dashboard
